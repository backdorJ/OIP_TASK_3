	<h1 id="firstHeading" class="firstHeading mw-first-heading"><span class="mw-page-title-main">Молекула</span></h1>
	<div id="bodyContent" class="vector-body">
		<div id="siteSub" class="noprint">Материал из Википедии — свободной энциклопедии</div>
		<div id="contentSub"><div id="mw-content-subtitle"></div></div>
		<div id="contentSub2"></div>
		
		<div id="jump-to-nav"></div>
		<a class="mw-jump-link" href="#mw-head">Перейти к навигации</a>
		<a class="mw-jump-link" href="#searchInput">Перейти к поиску</a>
		<div id="mw-content-text" class="mw-body-content"><div class="mw-content-ltr mw-parser-output" lang="ru" dir="ltr"><p><b>Моле́кула</b> (<span class="lang"><a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA#Латынь_в_Новое_время" title="Латинский язык">новолат.</a>&#160;<span dir="auto" style="font-style: italic;" lang="neolat">molecula</span></span>, уменьшительное от <a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латинский язык">лат.</a>&#160;<span lang="la" style="font-style:italic;">moles</span>&#160;— масса<sup id="cite&#95;ref-1" class="reference"><a href="#cite_note-1"><span class="cite-bracket">&#91;</span>1<span class="cite-bracket">&#93;</span></a></sup>)&#160;— <a href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B7%D0%B0%D1%80%D1%8F%D0%B4" title="Электрический заряд">электрически нейтральная</a> частица, образованная из двух или более связанных <a href="/wiki/%D0%A5%D0%B8%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D0%B2%D1%8F%D0%B7%D1%8C" title="Химическая связь">химическими связями</a> <a href="/wiki/%D0%90%D1%82%D0%BE%D0%BC" title="Атом">атомов</a><sup id="cite&#95;ref-iupac&#95;2-0" class="reference"><a href="#cite_note-iupac-2"><span class="cite-bracket">&#91;</span>2<span class="cite-bracket">&#93;</span></a></sup><sup id="cite&#95;ref-3" class="reference"><a href="#cite_note-3"><span class="cite-bracket">&#91;</span>3<span class="cite-bracket">&#93;</span></a></sup><sup id="cite&#95;ref-4" class="reference"><a href="#cite_note-4"><span class="cite-bracket">&#91;</span>4<span class="cite-bracket">&#93;</span></a></sup><sup id="cite&#95;ref-5" class="reference"><a href="#cite_note-5"><span class="cite-bracket">&#91;</span>5<span class="cite-bracket">&#93;</span></a></sup><sup id="cite&#95;ref-6" class="reference"><a href="#cite_note-6"><span class="cite-bracket">&#91;</span>6<span class="cite-bracket">&#93;</span></a></sup><sup id="cite&#95;ref-7" class="reference"><a href="#cite_note-7"><span class="cite-bracket">&#91;</span>7<span class="cite-bracket">&#93;</span></a></sup>.
</p><p>В отдельных разделах <a href="/wiki/%D0%A4%D0%B8%D0%B7%D0%B8%D0%BA%D0%B0" title="Физика">физики</a> к молекулам причисляют также <a href="/wiki/%D0%9E%D0%B4%D0%BD%D0%BE%D0%B0%D1%82%D0%BE%D0%BC%D0%BD%D0%B0%D1%8F_%D0%BC%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0" title="Одноатомная молекула">одноатомные молекулы</a>, то есть свободные (химически не связанные) атомы (например, <a href="/wiki/%D0%91%D0%BB%D0%B0%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B5_%D0%B3%D0%B0%D0%B7%D1%8B" title="Благородные газы">инертных газов</a>, <a href="/wiki/%D0%A0%D1%82%D1%83%D1%82%D1%8C" title="Ртуть">ртути</a> и&#160;т.&#160;п.).
</p><p>Обычно подразумевается, что молекулы нейтральны (не несут электрических зарядов) и не несут неспаренных <a href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BD" title="Электрон">электронов</a> (все <a href="/wiki/%D0%92%D0%B0%D0%BB%D0%B5%D0%BD%D1%82%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Валентность">валентности</a> насыщены); заряженные молекулы называют молекулярными <a href="/wiki/%D0%98%D0%BE%D0%BD" title="Ион">ионами</a>, молекулы с <a href="/wiki/%D0%9C%D1%83%D0%BB%D1%8C%D1%82%D0%B8%D0%BF%D0%BB%D0%B5%D1%82%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Мультиплетность">мультиплетностью</a>, отличной от <a href="/wiki/%D0%95%D0%B4%D0%B8%D0%BD%D0%B8%D1%86%D0%B0" class="mw-disambig" title="Единица">единицы</a> (то есть с неспаренными электронами и ненасыщенными <a href="/wiki/%D0%92%D0%B0%D0%BB%D0%B5%D0%BD%D1%82%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Валентность">валентностями</a>),&#160;— <a href="/wiki/%D0%A1%D0%B2%D0%BE%D0%B1%D0%BE%D0%B4%D0%BD%D1%8B%D0%B5_%D1%80%D0%B0%D0%B4%D0%B8%D0%BA%D0%B0%D0%BB%D1%8B" title="Свободные радикалы">радикалами</a>.
</p><p>Молекулы относительно высокой <a href="/wiki/%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D1%8F%D1%80%D0%BD%D0%B0%D1%8F_%D0%BC%D0%B0%D1%81%D1%81%D0%B0" title="Молекулярная масса">молекулярной массы</a>, состоящие из повторяющихся <a href="/w/index.php?title=%D0%9D%D0%B8%D0%B7%D0%BA%D0%BE%D0%BC%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D1%8F%D1%80%D0%BD%D1%8B%D0%B5_%D1%84%D1%80%D0%B0%D0%B3%D0%BC%D0%B5%D0%BD%D1%82%D1%8B&amp;action=edit&amp;redlink=1" class="new" title="Низкомолекулярные фрагменты (страница отсутствует)">низкомолекулярных фрагментов</a>, называются <a href="/wiki/%D0%9C%D0%B0%D0%BA%D1%80%D0%BE%D0%BC%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0" title="Макромолекула">макромолекулами</a><sup id="cite&#95;ref-8" class="reference"><a href="#cite_note-8"><span class="cite-bracket">&#91;</span>8<span class="cite-bracket">&#93;</span></a></sup>.
</p><p>С точки зрения квантовой механики<sup id="cite&#95;ref-9" class="reference"><a href="#cite_note-9"><span class="cite-bracket">&#91;</span>9<span class="cite-bracket">&#93;</span></a></sup> молекула представляет собой систему не из атомов, а из электронов и атомных ядер, взаимодействующих между собой.
</p><p>Особенности строения молекул определяют физические свойства <a href="/wiki/%D0%92%D0%B5%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Вещество">вещества</a>, состоящего из этих молекул.
</p><p>К веществам, сохраняющим молекулярную структуру в твёрдом состоянии, относятся, например, <a href="/wiki/%D0%92%D0%BE%D0%B4%D0%B0" title="Вода">вода</a>, <a href="/wiki/%D0%94%D0%B8%D0%BE%D0%BA%D1%81%D0%B8%D0%B4_%D1%83%D0%B3%D0%BB%D0%B5%D1%80%D0%BE%D0%B4%D0%B0" title="Диоксид углерода">оксид углерода(IV)</a>, многие органические вещества. Они характеризуются низкими температурами плавления и кипения. Большинство же твёрдых (кристаллических) неорганических веществ состоят не из молекул, а из ионов или атомов и существуют в виде макротел (кристалл <a href="/wiki/%D0%A5%D0%BB%D0%BE%D1%80%D0%B8%D0%B4_%D0%BD%D0%B0%D1%82%D1%80%D0%B8%D1%8F" title="Хлорид натрия">хлорида натрия</a>, кусок <a href="/wiki/%D0%9C%D0%B5%D0%B4%D1%8C" title="Медь">меди</a> и&#160;т.&#160;д.).
</p><p>Состав молекул сложных веществ выражается при помощи <a href="/wiki/%D0%A5%D0%B8%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%84%D0%BE%D1%80%D0%BC%D1%83%D0%BB%D0%B0" title="Химическая формула">химических формул</a>.
</p>

<div class="mw-heading mw-heading2"><h2 id="История"><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F"></span>История</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;veaction=edit&amp;section=1" title="Редактировать раздел «История»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;action=edit&amp;section=1" title="Редактировать код раздела «История»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>На международном <a href="/wiki/%D0%A1%D1%8A%D0%B5%D0%B7%D0%B4_%D1%85%D0%B8%D0%BC%D0%B8%D0%BA%D0%BE%D0%B2_%D0%B2_%D0%9A%D0%B0%D1%80%D0%BB%D1%81%D1%80%D1%83%D1%8D" title="Съезд химиков в Карлсруэ">съезде химиков в Карлсруэ</a> в <a href="/wiki/1860_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BD%D0%B0%D1%83%D0%BA%D0%B5" title="1860 год в науке">1860 году</a> были приняты определения понятий молекулы и атома. Молекула&#160;была определена как наименьшая частица химического вещества, обладающая всеми его химическими свойствами<sup id="cite&#95;ref-10" class="reference"><a href="#cite_note-10"><span class="cite-bracket">&#91;</span>10<span class="cite-bracket">&#93;</span></a></sup>.
</p>
<div class="mw-heading mw-heading3"><h3 id="Классическая_теория_химического_строения"><span id=".D0.9A.D0.BB.D0.B0.D1.81.D1.81.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B0.D1.8F_.D1.82.D0.B5.D0.BE.D1.80.D0.B8.D1.8F_.D1.85.D0.B8.D0.BC.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.BE.D0.B3.D0.BE_.D1.81.D1.82.D1.80.D0.BE.D0.B5.D0.BD.D0.B8.D1.8F"></span>Классическая теория химического строения</h3><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Классическая теория химического строения»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;action=edit&amp;section=2" title="Редактировать код раздела «Классическая теория химического строения»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>

<p>В классической теории химического строения молекула рассматривается как наименьшая стабильная частица вещества, обладающая всеми его химическими свойствами. В этом определении к молекулам относятся и одноатомные частицы (в частности, молекулы <a href="/wiki/%D0%91%D0%BB%D0%B0%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B5_%D0%B3%D0%B0%D0%B7%D1%8B" title="Благородные газы">инертных газов</a>).
</p><p>Молекула данного вещества имеет постоянный состав, то есть одинаковое количество атомов, объединённых <a href="/wiki/%D0%A5%D0%B8%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D0%B2%D1%8F%D0%B7%D1%8C" title="Химическая связь">химическими связями</a>, при этом химическая индивидуальность молекулы определяется именно совокупностью и конфигурацией химических связей, то есть <a href="/wiki/%D0%92%D0%B0%D0%BB%D0%B5%D0%BD%D1%82%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Валентность">валентными</a> взаимодействиями между входящими в её состав атомами, обеспечивающими её стабильность и основные свойства в достаточно широком диапазоне внешних условий. Невалентные взаимодействия (например, <a href="/wiki/%D0%92%D0%BE%D0%B4%D0%BE%D1%80%D0%BE%D0%B4%D0%BD%D0%B0%D1%8F_%D1%81%D0%B2%D1%8F%D0%B7%D1%8C" title="Водородная связь">водородные связи</a>), которые зачастую могут существенно влиять на свойства молекул и вещества, образуемого ими, в качестве критерия индивидуальности молекулы не учитываются.
</p><p>Центральным положением классической теории является положение о химической связи, при этом допускается наличие не только двухцентровых связей, объединяющих пары атомов, но и наличие многоцентровых (обычно трёхцентровых, иногда&#160;— четырёхцентровых) связей с «мостиковыми» атомами&#160;— как, например, мостиковых атомов водорода в <a href="/wiki/%D0%91%D0%BE%D1%80%D0%BE%D0%B2%D0%BE%D0%B4%D0%BE%D1%80%D0%BE%D0%B4%D1%8B" title="Бороводороды">боранах</a>, природа химической связи в классической теории не рассматривается&#160;— учитываются лишь такие интегральные характеристики, как <a href="/wiki/%D0%92%D0%B0%D0%BB%D0%B5%D0%BD%D1%82%D0%BD%D1%8B%D0%B9_%D1%83%D0%B3%D0%BE%D0%BB" title="Валентный угол">валентные углы</a>, диэдральные углы (углы между плоскостями, образованными тройками ядер), длины связей и <a href="/wiki/%D0%AD%D0%BD%D0%B5%D1%80%D0%B3%D0%B8%D1%8F_%D1%81%D0%B2%D1%8F%D0%B7%D0%B8" title="Энергия связи">их энергии</a>.
</p><p>Таким образом, молекула в классической теории представляется динамической системой, в которой атомы рассматриваются как материальные точки и в которой атомы и связанные группы атомов могут совершать механические вращательные и колебательные движения относительно некоторой равновесной ядерной конфигурации, соответствующей минимуму энергии молекулы и рассматривается как система <a href="/wiki/%D0%93%D0%B0%D1%80%D0%BC%D0%BE%D0%BD%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BE%D1%81%D1%86%D0%B8%D0%BB%D0%BB%D1%8F%D1%82%D0%BE%D1%80" title="Гармонический осциллятор">гармонических осцилляторов</a>.
</p><p>Молекула состоит из атомов, а если точнее, то из атомных ядер, окружённых определённым числом внутренних электронов, и внешних валентных электронов, образующих химические связи. Внутренние электроны атомов обычно не участвуют в образовании химических связей. Состав и строение молекул вещества не зависят от способа его получения.
</p><p>Атомы объединяются в молекуле в большинстве случаев с помощью химических связей. Как правило, такая связь образуется одной, двумя или тремя парами электронов, находящихся в совместном владении двух атомов, образуя общее электронное облако, форма которого описывается типом гибридизации. Молекула может иметь положительно и отрицательно заряженные атомы (<a href="/wiki/%D0%98%D0%BE%D0%BD" title="Ион">ионы</a>).
</p><p>Состав молекулы передаётся химическими формулами. Эмпирическая формула устанавливается на основе атомного соотношения элементов вещества и <a href="/wiki/%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D1%8F%D1%80%D0%BD%D0%B0%D1%8F_%D0%BC%D0%B0%D1%81%D1%81%D0%B0" title="Молекулярная масса">молекулярной массы</a>.
</p><p>Геометрическая структура молекулы определяется равновесным расположением атомных ядер. Энергия взаимодействия атомов зависит от расстояния между ядрами. На очень больших расстояниях эта энергия равна нулю. Если при сближении атомов образуется химическая связь, то атомы сильно притягиваются друг к другу (слабое притяжение наблюдается и без образования химической связи), при дальнейшем сближении начинают действовать <a href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D1%81%D1%82%D0%B0%D1%82%D0%B8%D0%BA%D0%B0" title="Электростатика">электростатические</a> силы отталкивания атомных ядер. Препятствием к сильному сближению атомов является также невозможность совмещения их внутренних электронных оболочек.
</p><p>Каждому атому в определённом валентном состоянии в молекуле можно приписать определённый атомный или ковалентный радиус (в случае ионной связи&#160;— ионный радиус), который характеризует размеры электронной оболочки атома (иона) образующего химическую связь в молекуле. Размер электронной оболочки молекулы является условной величиной. Существует вероятность (хотя и очень малая) найти электроны молекулы и на большем расстоянии от её атомного ядра. Практические размеры молекулы определяются равновесным расстоянием, на которое они могут быть сближены при плотной упаковке молекул в <a href="/wiki/%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D1%8F%D1%80%D0%BD%D1%8B%D0%B9_%D0%BA%D1%80%D0%B8%D1%81%D1%82%D0%B0%D0%BB%D0%BB" title="Молекулярный кристалл">молекулярном кристалле</a> и в <a href="/wiki/%D0%96%D0%B8%D0%B4%D0%BA%D0%BE%D1%81%D1%82%D1%8C" title="Жидкость">жидкости</a>. На бо́льших расстояниях молекулы притягиваются друг к другу, на меньших&#160;— отталкиваются. Длины связей в молекуле и ее размеры можно определить с помощью <a href="/wiki/%D0%A0%D0%B5%D0%BD%D1%82%D0%B3%D0%B5%D0%BD%D0%BE%D1%81%D1%82%D1%80%D1%83%D0%BA%D1%82%D1%83%D1%80%D0%BD%D1%8B%D0%B9_%D0%B0%D0%BD%D0%B0%D0%BB%D0%B8%D0%B7" title="Рентгеноструктурный анализ">рентгеноструктурного анализа</a> молекулярных кристаллов. Порядок величины этих размеров может быть определён из коэффициентов диффузии, теплопроводности и вязкости газов и с плотности вещества в конденсированном состоянии. Расстояние, на которое могут сблизиться валентно не связанные атомы одного и того же или разных молекул, может быть охарактеризована средними значениями так называемых <a href="/wiki/%D0%A0%D0%B0%D0%B4%D0%B8%D1%83%D1%81_%D0%92%D0%B0%D0%BD-%D0%B4%D0%B5%D1%80-%D0%92%D0%B0%D0%B0%D0%BB%D1%8C%D1%81%D0%B0" title="Радиус Ван-дер-Ваальса">ван-дер-ваальсовых радиусов</a>.
</p><p>Радиус Ван-дер-Ваальса существенно превышает ковалентный. Зная величины ван-дер-ваальсовых, ковалентных и ионных радиусов, можно построить наглядные модели молекул, которые бы отражали форму и размеры их электронных оболочек.
</p><p>Ковалентные химические связи в молекуле расположены под определёнными углами, которые зависят от состояния гибридизации атомных орбиталей. Так, для молекул насыщенных органических соединений характерно тетраэдральное (четырёхгранное) расположение связей, образуемых атомом углерода, для молекул с <a href="/wiki/%D0%94%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0%D1%8F_%D1%81%D0%B2%D1%8F%D0%B7%D1%8C_(%D1%85%D0%B8%D0%BC%D0%B8%D1%8F)" class="mw-redirect" title="Двойная связь (химия)">двойной связью</a> (<span class="nowrap">С=С</span>)&#160;— плоское расположение атомов углерода, для молекул соединений с <a href="/wiki/%D0%A2%D1%80%D0%BE%D0%B9%D0%BD%D0%B0%D1%8F_%D1%81%D0%B2%D1%8F%D0%B7%D1%8C" title="Тройная связь">тройной связью</a> (<span class="nowrap">С≡С</span>)&#160;— линейное расположение связей. Таким образом, многоатомная молекула имеет определённую конфигурацию в пространстве, то есть определённую геометрию расположения связей, которая не может быть изменена без их разрыва. Молекула характеризуется той или иной симметрией расположения атомов. Если молекула не имеет плоскости и центра симметрии, то она может существовать в двух конфигурациях, которые представляют собой зеркальные отражения друг друга (зеркальные антиподы, оптические изомеры или <a href="/wiki/%D0%AD%D0%BD%D0%B0%D0%BD%D1%82%D0%B8%D0%BE%D0%BC%D0%B5%D1%80%D1%8B" title="Энантиомеры">энантиомеры</a>). Все важнейшие биологические функциональные вещества в живой природе существуют в форме одного определённого энантиомера.
</p><p>Молекулы, содержащие одинарные связи (<a href="/wiki/%D0%A1%D0%B8%D0%B3%D0%BC%D0%B0-%D1%81%D0%B2%D1%8F%D0%B7%D1%8C" title="Сигма-связь">сигма-связи</a>), не  включённые в трехчленные циклы, могут существовать в различных конформациях, возникающих при поворотах атомных групп или фрагментов молекул вокруг этих связей. Важные особенности свойств  макромолекул синтетических и биологических <a href="/wiki/%D0%9F%D0%BE%D0%BB%D0%B8%D0%BC%D0%B5%D1%80%D1%8B" title="Полимеры">полимеров</a> определяются именно их конформационными свойствами.
</p>
<div class="mw-heading mw-heading3"><h3 id="Квантохимическая_теория_химического_строения"><span id=".D0.9A.D0.B2.D0.B0.D0.BD.D1.82.D0.BE.D1.85.D0.B8.D0.BC.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B0.D1.8F_.D1.82.D0.B5.D0.BE.D1.80.D0.B8.D1.8F_.D1.85.D0.B8.D0.BC.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.BE.D0.B3.D0.BE_.D1.81.D1.82.D1.80.D0.BE.D0.B5.D0.BD.D0.B8.D1.8F"></span>Квантохимическая теория химического строения</h3><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Квантохимическая теория химического строения»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;action=edit&amp;section=3" title="Редактировать код раздела «Квантохимическая теория химического строения»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>В квантохимической теории химического строения основными параметрами, определяющими индивидуальность молекулы, является её <a href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BD%D0%BD%D0%B0%D1%8F_%D0%BA%D0%BE%D0%BD%D1%84%D0%B8%D0%B3%D1%83%D1%80%D0%B0%D1%86%D0%B8%D1%8F" title="Электронная конфигурация">электронная</a> и <a href="/wiki/%D0%98%D0%B7%D0%BE%D0%BC%D0%B5%D1%80#Пространственная_изомерия_(стереоизомерия)" class="mw-redirect" title="Изомер">пространственная (стереохимическая)</a> конфигурации. При этом, в качестве электронной конфигурации, определяющей свойства молекулы, принимается конфигурация с наинизшей энергией, то есть основное энергетическое состояние.
</p>
<div class="mw-heading mw-heading2"><h2 id="Представление_структуры_молекул"><span id=".D0.9F.D1.80.D0.B5.D0.B4.D1.81.D1.82.D0.B0.D0.B2.D0.BB.D0.B5.D0.BD.D0.B8.D0.B5_.D1.81.D1.82.D1.80.D1.83.D0.BA.D1.82.D1.83.D1.80.D1.8B_.D0.BC.D0.BE.D0.BB.D0.B5.D0.BA.D1.83.D0.BB"></span>Представление структуры молекул</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Представление структуры молекул»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;action=edit&amp;section=4" title="Редактировать код раздела «Представление структуры молекул»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Молекулы состоят из электронов и атомных ядер, расположение последних в молекуле отражает <a href="/wiki/%D0%A1%D1%82%D1%80%D1%83%D0%BA%D1%82%D1%83%D1%80%D0%BD%D0%B0%D1%8F_%D1%84%D0%BE%D1%80%D0%BC%D1%83%D0%BB%D0%B0" title="Структурная формула">структурная формула</a> (для передачи состава используется так называемая <a href="/wiki/%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D1%8F%D1%80%D0%BD%D0%B0%D1%8F_%D1%84%D0%BE%D1%80%D0%BC%D1%83%D0%BB%D0%B0" title="Молекулярная формула">молекулярная формула</a>). Молекулы белков и некоторых искусственно синтезированных соединений могут содержать сотни тысяч атомов. Отдельно рассматриваются <a href="/wiki/%D0%9C%D0%B0%D0%BA%D1%80%D0%BE%D0%BC%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0" title="Макромолекула">макромолекулы</a> полимеров.
</p><p>Молекулы являются объектом изучения теории <a href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%BE%D0%B5%D0%BD%D0%B8%D0%B5_%D0%BC%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB&amp;action=edit&amp;redlink=1" class="new" title="Строение молекул (страница отсутствует)">строения молекул</a>, <a href="/wiki/%D0%9A%D0%B2%D0%B0%D0%BD%D1%82%D0%BE%D0%B2%D0%B0%D1%8F_%D1%85%D0%B8%D0%BC%D0%B8%D1%8F" title="Квантовая химия">квантовой химии</a>, аппарат которых активно использует достижения <a href="/wiki/%D0%9A%D0%B2%D0%B0%D0%BD%D1%82%D0%BE%D0%B2%D0%B0%D1%8F_%D1%84%D0%B8%D0%B7%D0%B8%D0%BA%D0%B0" title="Квантовая физика">квантовой физики</a>, в том числе релятивистских её разделов. Также в настоящее время развивается такая область химии, как <a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D1%8F%D1%80%D0%BD%D1%8B%D0%B9_%D0%B4%D0%B8%D0%B7%D0%B0%D0%B9%D0%BD&amp;action=edit&amp;redlink=1" class="new" title="Молекулярный дизайн (страница отсутствует)">молекулярный дизайн</a>. Для определения строения молекул конкретного вещества современная наука располагает широким набором методов: <a href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BD%D0%BD%D0%B0%D1%8F_%D1%81%D0%BF%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D1%81%D0%BA%D0%BE%D0%BF%D0%B8%D1%8F" title="Электронная спектроскопия">электронная спектроскопия</a>, <a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%B5%D0%B1%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D1%81%D0%BF%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D1%81%D0%BA%D0%BE%D0%BF%D0%B8%D1%8F" class="mw-redirect" title="Колебательная спектроскопия">колебательная спектроскопия</a>, <a href="/wiki/%D0%AF%D0%B4%D0%B5%D1%80%D0%BD%D1%8B%D0%B9_%D0%BC%D0%B0%D0%B3%D0%BD%D0%B8%D1%82%D0%BD%D1%8B%D0%B9_%D1%80%D0%B5%D0%B7%D0%BE%D0%BD%D0%B0%D0%BD%D1%81" title="Ядерный магнитный резонанс">ядерный магнитный резонанс</a> и <a href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BD%D0%BD%D1%8B%D0%B9_%D0%BF%D0%B0%D1%80%D0%B0%D0%BC%D0%B0%D0%B3%D0%BD%D0%B8%D1%82%D0%BD%D1%8B%D0%B9_%D1%80%D0%B5%D0%B7%D0%BE%D0%BD%D0%B0%D0%BD%D1%81" title="Электронный парамагнитный резонанс">электронный парамагнитный резонанс</a> и многие другие, но единственными прямыми методами в настоящее время являются дифракционные методы, а именно: <a href="/wiki/%D0%A0%D0%B5%D0%BD%D1%82%D0%B3%D0%B5%D0%BD%D0%BE%D1%81%D1%82%D1%80%D1%83%D0%BA%D1%82%D1%83%D1%80%D0%BD%D1%8B%D0%B9_%D0%B0%D0%BD%D0%B0%D0%BB%D0%B8%D0%B7" title="Рентгеноструктурный анализ">рентгеноструктурный анализ</a> и <a href="/wiki/%D0%94%D0%B8%D1%84%D1%80%D0%B0%D0%BA%D1%86%D0%B8%D1%8F_%D0%BD%D0%B5%D0%B9%D1%82%D1%80%D0%BE%D0%BD%D0%BE%D0%B2" class="mw-redirect" title="Дифракция нейтронов">дифракция нейтронов</a>.
</p>
<div class="mw-heading mw-heading2"><h2 id="Взаимодействие_атомов_при_образовании_молекулы"><span id=".D0.92.D0.B7.D0.B0.D0.B8.D0.BC.D0.BE.D0.B4.D0.B5.D0.B9.D1.81.D1.82.D0.B2.D0.B8.D0.B5_.D0.B0.D1.82.D0.BE.D0.BC.D0.BE.D0.B2_.D0.BF.D1.80.D0.B8_.D0.BE.D0.B1.D1.80.D0.B0.D0.B7.D0.BE.D0.B2.D0.B0.D0.BD.D0.B8.D0.B8_.D0.BC.D0.BE.D0.BB.D0.B5.D0.BA.D1.83.D0.BB.D1.8B"></span>Взаимодействие атомов при образовании молекулы</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Взаимодействие атомов при образовании молекулы»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;action=edit&amp;section=5" title="Редактировать код раздела «Взаимодействие атомов при образовании молекулы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Природа химических связей в молекуле оставалась загадкой до создания <a href="/wiki/%D0%9A%D0%B2%D0%B0%D0%BD%D1%82%D0%BE%D0%B2%D0%B0%D1%8F_%D0%BC%D0%B5%D1%85%D0%B0%D0%BD%D0%B8%D0%BA%D0%B0" title="Квантовая механика">квантовой механики</a>&#160;— классическая физика не могла объяснить насыщаемость и направленность валентных связей. Основы теории химической связи были заложены в 1927 году <a href="/wiki/%D0%93%D0%B0%D0%B9%D1%82%D0%BB%D0%B5%D1%80,_%D0%92%D0%B0%D0%BB%D1%8C%D1%82%D0%B5%D1%80" title="Гайтлер, Вальтер">Вальтером Гайтлером</a> и Фрицем Лондоном на примере простейшей молекулы Н<sub>2</sub>. Позже теория и методы расчётов были значительно усовершенствованы.
</p><p>Химические связи в молекулах подавляющего большинства органических соединений является ковалентными. Среди неорганических соединений существуют ионные и донорно-акцепторные связи, которые реализуются в результате обобществления пары электронов атома. Энергия образования молекулы из атомов во многих рядах подобных соединений приближённо аддитивна. То есть можно считать, что энергия молекулы&#160;— это сумма энергий её связей, имеющих постоянные значения в таких рядах.
</p><p>Аддитивность энергии молекулы выполняется не всегда. Примером нарушения аддитивности являются плоские молекулы органических соединений с так называемыми сопряжёнными связями, то есть с кратными связями, которые чередуются с одинарными. Сильная делокализация <i>p</i>-состояний электронов приводит к стабилизации молекулы. Выравнивание электронной плотности вследствие коллективизации <style data-mw-deduplicate="TemplateStyles:r117753614">.mw-parser-output .ts-math{white-space:nowrap;font-family:times,serif,palatino linotype,new athena unicode,athena,gentium,code2000;font-size:120%}</style><span class="ts-math"><i>p</i></span>-состояний электронов по связям выражается в укорочении двойных связей и удлинении одинарных. В <a href="/wiki/%D0%91%D0%B5%D0%BD%D0%B7%D0%BE%D0%BB" title="Бензол">бензоле</a> углерод–углеродные связи одинаковы, образуют правильный <a href="/wiki/%D0%A8%D0%B5%D1%81%D1%82%D0%B8%D1%83%D0%B3%D0%BE%D0%BB%D1%8C%D0%BD%D0%B8%D0%BA" title="Шестиугольник">шестиугольник</a> и имеют длину, среднюю между длиной одинарной и двойной связи. Сопряжение связей ярко проявляется в молекулярных спектрах. Современная квантовомеханическая теория химических связей учитывает делокализации не только <link rel="mw-deduplicated-inline-style" href="mw-data:TemplateStyles:r117753614" /><span class="ts-math"><i>p</i></span>-, но и <link rel="mw-deduplicated-inline-style" href="mw-data:TemplateStyles:r117753614" /><span class="ts-math"><i>s</i></span>-состояний электронов, которая наблюдается в любых молекулах.
</p><p>В подавляющем большинстве случаев суммарный <a href="/wiki/%D0%A1%D0%BF%D0%B8%D0%BD" title="Спин">спин</a> валентных электронов в молекуле равен нулю. Молекулы, содержащие неспаренные электроны&#160;— <a href="/wiki/%D0%A1%D0%B2%D0%BE%D0%B1%D0%BE%D0%B4%D0%BD%D1%8B%D0%B5_%D1%80%D0%B0%D0%B4%D0%B8%D0%BA%D0%B0%D0%BB%D1%8B" title="Свободные радикалы">свободные радикалы</a> (например, атомарный водород Н, метил ·CH<sub>3</sub>), обычно неустойчивы, поскольку при их взаимодействии друг с другом происходит значительное снижение энергии вследствие образования <a href="/wiki/%D0%9A%D0%BE%D0%B2%D0%B0%D0%BB%D0%B5%D0%BD%D1%82%D0%BD%D0%B0%D1%8F_%D1%81%D0%B2%D1%8F%D0%B7%D1%8C" title="Ковалентная связь">ковалентных связей</a>. Они могут устойчиво существовать при таких температурах, когда средняя кинетическая энергия молекулы превосходит или сравнима с энергией связи, но при этом ниже энергии разрушения (например ионизации) радикала.
</p>
<div class="mw-heading mw-heading2"><h2 id="Электрические_и_оптические_свойства_молекул"><span id=".D0.AD.D0.BB.D0.B5.D0.BA.D1.82.D1.80.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B8.D0.B5_.D0.B8_.D0.BE.D0.BF.D1.82.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B8.D0.B5_.D1.81.D0.B2.D0.BE.D0.B9.D1.81.D1.82.D0.B2.D0.B0_.D0.BC.D0.BE.D0.BB.D0.B5.D0.BA.D1.83.D0.BB"></span>Электрические и оптические свойства молекул</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Электрические и оптические свойства молекул»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;action=edit&amp;section=6" title="Редактировать код раздела «Электрические и оптические свойства молекул»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Поведение вещества в электрическом поле определяется основными электрическими характеристиками молекул&#160;— постоянным дипольным моментом и поляризуемостью.
</p><p>Дипольный момент означает несовпадение «центров тяжести» положительных и отрицательных зарядов в молекуле (электрическую асимметрию молекулы). То есть молекулы, имеющие центр симметрии, например H<sub>2</sub>, лишены постоянного дипольного момента.
</p><p>Поляризуемость&#160;— это способность электронной оболочки любой молекулы перемещаться под действием электрического поля, в результате чего в молекуле образуется наведённый дипольный момент. Значение дипольного момента и поляризуемости определяют экспериментально с помощью измерения диэлектрической проницаемости.
</p><p>Оптические свойства вещества характеризуют его поведение в переменном <a href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B5_%D0%BF%D0%BE%D0%BB%D0%B5" title="Электрическое поле">электрическом поле</a> световой волны и определяются поляризуемостью молекулы этого вещества. С поляризуемостью непосредственно связаны преломление и рассеяние света, оптическая активность и другие явления, изучаемые молекулярной оптикой.
</p>
<div class="mw-heading mw-heading2"><h2 id="Межмолекулярное_взаимодействие"><span id=".D0.9C.D0.B5.D0.B6.D0.BC.D0.BE.D0.BB.D0.B5.D0.BA.D1.83.D0.BB.D1.8F.D1.80.D0.BD.D0.BE.D0.B5_.D0.B2.D0.B7.D0.B0.D0.B8.D0.BC.D0.BE.D0.B4.D0.B5.D0.B9.D1.81.D1.82.D0.B2.D0.B8.D0.B5"></span>Межмолекулярное взаимодействие</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Межмолекулярное взаимодействие»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;action=edit&amp;section=7" title="Редактировать код раздела «Межмолекулярное взаимодействие»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<div role="note" class="hatnote navigation-not-searchable ts-main"><style data-mw-deduplicate="TemplateStyles:r142002967">.mw-parser-output .ts-main a{font-weight:bold}.mw-parser-output .ts-main a.new,.mw-parser-output .ts-main a.extiw,.mw-parser-output .ts-main a.external{font-weight:normal}</style>Основная статья: <a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%BC%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D1%8F%D1%80%D0%BD%D0%BE%D0%B5_%D0%B2%D0%B7%D0%B0%D0%B8%D0%BC%D0%BE%D0%B4%D0%B5%D0%B9%D1%81%D1%82%D0%B2%D0%B8%D0%B5" title="Межмолекулярное взаимодействие">Межмолекулярное взаимодействие</a></div>
<p>Межмолекулярное взаимодействие&#160;— взаимодействие между электрически нейтральными молекулами в пространстве. В зависимости от <a href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8F%D1%80%D0%BD%D0%BE%D1%81%D1%82%D1%8C_%D1%85%D0%B8%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D1%85_%D1%81%D0%B2%D1%8F%D0%B7%D0%B5%D0%B9" title="Полярность химических связей">полярности молекул</a> характер межмолекулярного взаимодействия разный. Природа последнего оставалась до конца неясной до создания квантовой механики.
</p>
<figure class="mw-default-size" typeof="mw:File/Thumb"><a href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Dipoli_acqua.png" class="mw-file-description"><img src="//upload.wikimedia.org/wikipedia/commons/thumb/8/89/Dipoli_acqua.png/250px-Dipoli_acqua.png" decoding="async" width="250" height="210" class="mw-file-element" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/89/Dipoli_acqua.png/500px-Dipoli_acqua.png 1.5x" data-file-width="712" data-file-height="599" /></a><figcaption> Молекула воды, пример <a href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8F%D1%80%D0%BD%D1%8B%D0%B5_%D0%B2%D0%B5%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%B0" title="Полярные вещества">поляризованной молекулы</a>. <p>Числами указана <a href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BE%D1%82%D1%80%D0%B8%D1%86%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Электроотрицательность">электроотрицательность</a>&#160;— свойство атома в молекуле смещать к себе общие <a href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BD%D0%BD%D1%8B%D0%B5_%D0%BF%D0%B0%D1%80%D1%8B" class="mw-redirect" title="Электронные пары">электронные пары</a>. Электроотрицательность атома кислорода (3,4) больше электроотрицательности атома водорода (2,2).</p></figcaption></figure>
<ol><li><b>Ориентационный тип межмолекулярного взаимодействия</b> возникает между двумя полярными молекулами, то есть, такими, которые имеют собственный дипольный момент. Взаимодействие дипольных моментов и определяет результирующую силу&#160;— притяжения или отталкивания. В случае, если дипольные моменты молекул размещаются на одной линии, взаимодействие молекул будет интенсивней.</li>
<li><b>Индукционный тип</b>&#160;— между одной полярной и одной неполярной молекулами. Полярная молекула поляризует неполярную, при этом фрагмент поляризуемой молекулы приобретает заряд, противоположный фрагменту молекулы поляризатора, взаимодействующего с ним. Это обусловливает поляризацию неполярной молекулы, то есть явления смещения связанной электронной оболочки относительно центра положительного заряда.</li>
<li><b>Дисперсионный</b>&#160;— между двумя неполярными молекулами. В общем, дипольные моменты неполярных молекул равны нулю, однако в определённый момент времени, есть вероятность распределения электронов по всему объёму молекулы неравномерно. Вследствие этого возникает <b>мгновенный дипольный момент</b>. При этом, мгновенный диполь или поляризует соседние неполярные молекулы, или взаимодействует с мгновенным диполем другой нейтральной молекулы.</li></ol>
<div class="mw-heading mw-heading2"><h2 id="Магнитные_свойства_молекул"><span id=".D0.9C.D0.B0.D0.B3.D0.BD.D0.B8.D1.82.D0.BD.D1.8B.D0.B5_.D1.81.D0.B2.D0.BE.D0.B9.D1.81.D1.82.D0.B2.D0.B0_.D0.BC.D0.BE.D0.BB.D0.B5.D0.BA.D1.83.D0.BB"></span>Магнитные свойства молекул</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;veaction=edit&amp;section=8" title="Редактировать раздел «Магнитные свойства молекул»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;action=edit&amp;section=8" title="Редактировать код раздела «Магнитные свойства молекул»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Молекулы и макромолекулы подавляющего большинства химических соединений являются <a href="/wiki/%D0%94%D0%B8%D0%B0%D0%BC%D0%B0%D0%B3%D0%BD%D0%B5%D1%82%D0%B8%D0%BA%D0%B8" title="Диамагнетики">диамагнитными</a>. <a href="/wiki/%D0%9C%D0%B0%D0%B3%D0%BD%D0%B8%D1%82%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D1%81%D0%BF%D1%80%D0%B8%D0%B8%D0%BC%D1%87%D0%B8%D0%B2%D0%BE%D1%81%D1%82%D1%8C" title="Магнитная восприимчивость">Магнитная восприимчивость</a> молекул (<link rel="mw-deduplicated-inline-style" href="mw-data:TemplateStyles:r117753614" /><span class="ts-math">χ</span>) для отдельных органических соединений может быть выражена как сумма значений <link rel="mw-deduplicated-inline-style" href="mw-data:TemplateStyles:r117753614" /><span class="ts-math">χ</span> для отдельных связей.
</p><p>Молекулы, имеющие постоянный магнитный момент, является <a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B0%D0%BC%D0%B0%D0%B3%D0%BD%D0%B5%D1%82%D0%B8%D0%BA%D0%B8" title="Парамагнетики">парамагнитными</a>. К таковым относятся молекулы с нечётным количеством электронов на внешней оболочке (например, <a href="/wiki/%D0%9E%D0%BA%D1%81%D0%B8%D0%B4_%D0%B0%D0%B7%D0%BE%D1%82%D0%B0(II)" title="Оксид азота(II)">NO</a> и любые свободные радикалы), молекулы, содержащие атомы с незаполненными внутренними оболочками (например, <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B5%D1%85%D0%BE%D0%B4%D0%BD%D1%8B%D0%B5_%D0%BC%D0%B5%D1%82%D0%B0%D0%BB%D0%BB%D1%8B" title="Переходные металлы">переходные металлы</a>). Магнитная восприимчивость парамагнитных веществ зависит от температуры, поскольку тепловое движение препятствует ориентации магнитных моментов в <a href="/wiki/%D0%9C%D0%B0%D0%B3%D0%BD%D0%B8%D1%82%D0%BD%D0%BE%D0%B5_%D0%BF%D0%BE%D0%BB%D0%B5" title="Магнитное поле">магнитном поле</a>.
</p>
<div class="mw-heading mw-heading2"><h2 id="Спектры_и_строение_молекул"><span id=".D0.A1.D0.BF.D0.B5.D0.BA.D1.82.D1.80.D1.8B_.D0.B8_.D1.81.D1.82.D1.80.D0.BE.D0.B5.D0.BD.D0.B8.D0.B5_.D0.BC.D0.BE.D0.BB.D0.B5.D0.BA.D1.83.D0.BB"></span>Спектры и строение молекул</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;veaction=edit&amp;section=9" title="Редактировать раздел «Спектры и строение молекул»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;action=edit&amp;section=9" title="Редактировать код раздела «Спектры и строение молекул»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Электрические, оптические, магнитные и другие свойства молекул связаны с волновыми функциями и энергиями различных состояний молекул. Информацию о состояниях молекул и вероятности перехода между ними дают молекулярные спектры.
</p><p>Частоты колебаний в спектрах определяются массами атомов, их расположением и динамикой межатомных взаимодействий. Частоты в спектрах зависят от моментов инерции молекул, определение которых из <a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D1%8F%D1%80%D0%BD%D0%B0%D1%8F_%D1%81%D0%BF%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D1%81%D0%BA%D0%BE%D0%BF%D0%B8%D1%8F&amp;action=edit&amp;redlink=1" class="new" title="Молекулярная спектроскопия (страница отсутствует)">спектроскопических</a> данных позволяет получить точные значения межатомных расстояний в молекуле. Общее число линий и полос в колебательном спектре молекулы зависит от её симметрии.
</p><p>Электронные переходы в молекулах характеризуют структуру их <a href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BD%D0%BD%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BE%D0%BB%D0%BE%D1%87%D0%BA%D0%B0" title="Электронная оболочка">электронных оболочек</a> и состояние <a href="/wiki/%D0%A5%D0%B8%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D0%B2%D1%8F%D0%B7%D1%8C" title="Химическая связь">химических связей</a>. Спектры молекул, которые имеют большее количество связей, характеризуются длинноволновыми полосами поглощения, попадающими в видимую область. Вещества, которые построены из таких молекул, характеризуются окраской; к таким веществам относятся все органические красители.
</p>
<div class="mw-heading mw-heading2"><h2 id="Молекулы_в_химии,_физике_и_биологии"><span id=".D0.9C.D0.BE.D0.BB.D0.B5.D0.BA.D1.83.D0.BB.D1.8B_.D0.B2_.D1.85.D0.B8.D0.BC.D0.B8.D0.B8.2C_.D1.84.D0.B8.D0.B7.D0.B8.D0.BA.D0.B5_.D0.B8_.D0.B1.D0.B8.D0.BE.D0.BB.D0.BE.D0.B3.D0.B8.D0.B8"></span>Молекулы в химии, физике и биологии</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;veaction=edit&amp;section=10" title="Редактировать раздел «Молекулы в химии, физике и биологии»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0&amp;action=edit&amp;section=10" title="Редактировать код раздела «Молекулы в химии, физике и биологии»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Понятие молекулы является основным для химии, и большей частью сведений о строении и функциональность молекул наука обязана химическим исследованиям. Химия определяет строение молекул на основе химических реакций и, наоборот, на основе строения молекулы определяет, каким будет ход реакций.
</p><p>Строением и свойствами молекулы определяются физические явления, которые изучаются молекулярной физикой. В физике понятие молекулы используется для объяснения свойств газов, жидкостей и твёрдых тел. Подвижностью молекул определяется способность вещества к <a href="/wiki/%D0%94%D0%B8%D1%84%D1%84%D1%83%D0%B7%D0%B8%D1%8F" title="Диффузия">диффузии</a>, его <a href="/wiki/%D0%92%D1%8F%D0%B7%D0%BA%D0%BE%D1%81%D1%82%D1%8C" title="Вязкость">вязкость</a>, <a href="/wiki/%D0%A2%D0%B5%D0%BF%D0%BB%D0%BE%D0%BF%D1%80%D0%BE%D0%B2%D0%BE%D0%B4%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Теплопроводность">теплопроводность</a> и&#160;т.&#160;д. Первое прямое экспериментальное доказательство существования молекул было получено французским физиком <a href="/wiki/%D0%9F%D0%B5%D1%80%D1%80%D0%B5%D0%BD,_%D0%96%D0%B0%D0%BD_%D0%91%D0%B0%D1%82%D0%B8%D1%81%D1%82" title="Перрен, Жан Батист">Жаном Перреном</a> в <a href="/wiki/1906_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BD%D0%B0%D1%83%D0%BA%D0%B5" title="1906 год в науке">1906 году</a> при изучении <a href="/wiki/%D0%91%D1%80%D0%BE%D1%83%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%BE%D0%B5_%D0%B4%D0%B2%D0%B8%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="Броуновское движение">броуновского движения</a>.
</p><p>Поскольку все живые организмы существуют на основе тонко сбалансированного химического и нехимического взаимодействия между молекулами, изучение строения и свойств молекул имеет фундаментальное значение для биологии и естествознания в целом.
</p><p>Развитие биологии, химии и молекулярной физики привели к возникновению <a href="/wiki/%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D1%8F%D1%80%D0%BD%D0%B0%D1%8F_%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Молекулярная биология">молекулярной биологии</a>, которая исследует основные явления жизни, исходя из строения и свойств биологически функциональных молекул.
</p>
