	<h1 id="firstHeading" class="firstHeading mw-first-heading"><span class="mw-page-title-main">Атмосфера</span></h1>
	<div id="bodyContent" class="vector-body">
		<div id="siteSub" class="noprint">Материал из Википедии — свободной энциклопедии</div>

		<div id="mw-content-text" class="mw-body-content"><div class="mw-content-ltr mw-parser-output" lang="ru" dir="ltr"><div role="note" class="hatnote navigation-not-searchable dabhide">Эта статья&#160;— о&#160;газовой оболочке планет. Об&#160;атмосфере Земли см.&#160;<a href="/wiki/%D0%90%D1%82%D0%BC%D0%BE%D1%81%D1%84%D0%B5%D1%80%D0%B0_%D0%97%D0%B5%D0%BC%D0%BB%D0%B8" title="Атмосфера Земли">атмосфера Земли</a>&#x3b; о&#160;внешней оболочке звёзд см.&#160;<a href="/wiki/%D0%97%D0%B2%D1%91%D0%B7%D0%B4%D0%BD%D0%B0%D1%8F_%D0%B0%D1%82%D0%BC%D0%BE%D1%81%D1%84%D0%B5%D1%80%D0%B0" title="Звёздная атмосфера">Звёздная атмосфера</a>&#x3b; о&#160;других значения термина см.&#160;<a href="/wiki/%D0%90%D1%82%D0%BC%D0%BE%D1%81%D1%84%D0%B5%D1%80%D0%B0_(%D0%B7%D0%BD%D0%B0%D1%87%D0%B5%D0%BD%D0%B8%D1%8F)" class="mw-disambig" title="Атмосфера (значения)">Атмосфера (значения)</a>.</div>
<figure typeof="mw:File/Thumb"><a href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Top_of_Atmosphere.jpg" class="mw-file-description"><img src="//upload.wikimedia.org/wikipedia/commons/thumb/b/be/Top_of_Atmosphere.jpg/250px-Top_of_Atmosphere.jpg" decoding="async" width="250" height="166" class="mw-file-element" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/be/Top_of_Atmosphere.jpg/500px-Top_of_Atmosphere.jpg 1.5x" data-file-width="3027" data-file-height="2010" /></a><figcaption><a href="/wiki/%D0%90%D1%82%D0%BC%D0%BE%D1%81%D1%84%D0%B5%D1%80%D0%BD%D1%8B%D0%B5_%D0%B3%D0%B0%D0%B7%D1%8B" class="mw-redirect" title="Атмосферные газы">Атмосферные газы</a> Земли <a href="/wiki/%D0%94%D0%B8%D1%84%D1%84%D1%83%D0%B7%D0%BD%D0%BE%D0%B5_%D0%B8%D0%B7%D0%BB%D1%83%D1%87%D0%B5%D0%BD%D0%B8%D0%B5_%D0%BD%D0%B5%D0%B1%D0%B0" title="Диффузное излучение неба">рассеивают синие длины волн лучше</a> других, поэтому если смотреть из космоса, то вокруг нашей планеты имеется голубое <a href="/wiki/%D0%93%D0%B0%D0%BB%D0%BE" title="Гало">гало</a>, а если смотреть с Земли, то видно голубое небо.</figcaption></figure>
<p><b>Атмосфе́ра</b> (от <a href="/wiki/%D0%94%D1%80%D0%B5%D0%B2%D0%BD%D0%B5%D0%B3%D1%80%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Древнегреческий язык">др.-греч.</a> <span lang="grc"><span style="font-family: palatino linotype, new athena unicode, athena, gentium, code2000, serif; font-size: 105%;">ἀτμός</span></span>&#160;— «<a href="/wiki/%D0%9F%D0%B0%D1%80" title="Пар">пар</a>» и <span lang="grc"><span style="font-family: palatino linotype, new athena unicode, athena, gentium, code2000, serif; font-size: 105%;">σφαῖρα</span></span>&#160;— «<a href="/wiki/%D0%A1%D1%84%D0%B5%D1%80%D0%B0" title="Сфера">сфера</a>»)&#160;— <a href="/wiki/%D0%93%D0%B0%D0%B7" title="Газ">газовая</a> оболочка <a href="/wiki/%D0%9D%D0%B5%D0%B1%D0%B5%D1%81%D0%BD%D0%BE%D0%B5_%D1%82%D0%B5%D0%BB%D0%BE" class="mw-redirect" title="Небесное тело">небесного тела</a>, удерживаемая около него <a href="/wiki/%D0%93%D1%80%D0%B0%D0%B2%D0%B8%D1%82%D0%B0%D1%86%D0%B8%D1%8F" title="Гравитация">гравитацией</a>. Поскольку не существует резкой границы между атмосферой и <a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%BF%D0%BB%D0%B0%D0%BD%D0%B5%D1%82%D0%BD%D0%BE%D0%B5_%D0%BF%D1%80%D0%BE%D1%81%D1%82%D1%80%D0%B0%D0%BD%D1%81%D1%82%D0%B2%D0%BE" title="Межпланетное пространство">межпланетным пространством</a>, то обычно атмосферой принято считать область вокруг небесного тела, в которой газовая среда вращается вместе с ним как единое целое. Толщина атмосферы некоторых <a href="/wiki/%D0%9F%D0%BB%D0%B0%D0%BD%D0%B5%D1%82%D0%B0" title="Планета">планет</a>, состоящих в основном из газов (<a href="/wiki/%D0%93%D0%B0%D0%B7%D0%BE%D0%B2%D1%8B%D0%B5_%D0%BF%D0%BB%D0%B0%D0%BD%D0%B5%D1%82%D1%8B" class="mw-redirect" title="Газовые планеты">газовые планеты</a>), может быть очень большой.
</p><p><a href="/wiki/%D0%90%D1%82%D0%BC%D0%BE%D1%81%D1%84%D0%B5%D1%80%D0%B0_%D0%97%D0%B5%D0%BC%D0%BB%D0%B8" title="Атмосфера Земли">Атмосфера Земли</a> содержит <a href="/wiki/%D0%9A%D0%B8%D1%81%D0%BB%D0%BE%D1%80%D0%BE%D0%B4" title="Кислород">кислород</a>, используемый большинством живых организмов для <a href="/wiki/%D0%94%D1%8B%D1%85%D0%B0%D0%BD%D0%B8%D0%B5" title="Дыхание">дыхания</a>, и <a href="/wiki/%D0%94%D0%B8%D0%BE%D0%BA%D1%81%D0%B8%D0%B4_%D1%83%D0%B3%D0%BB%D0%B5%D1%80%D0%BE%D0%B4%D0%B0" title="Диоксид углерода">диоксид углерода</a>, потребляемый <a href="/wiki/%D0%A0%D0%B0%D1%81%D1%82%D0%B5%D0%BD%D0%B8%D1%8F" title="Растения">растениями</a> и <a href="/wiki/%D0%A6%D0%B8%D0%B0%D0%BD%D0%BE%D0%B1%D0%B0%D0%BA%D1%82%D0%B5%D1%80%D0%B8%D0%B8" title="Цианобактерии">цианобактериями</a> в процессе <a href="/wiki/%D0%A4%D0%BE%D1%82%D0%BE%D1%81%D0%B8%D0%BD%D1%82%D0%B5%D0%B7" title="Фотосинтез">фотосинтеза</a>. Атмосфера также является защитным слоем <a href="/wiki/%D0%9F%D0%BB%D0%B0%D0%BD%D0%B5%D1%82%D0%B0" title="Планета">планеты</a>, защищая её обитателей от солнечного <a href="/wiki/%D0%A3%D0%BB%D1%8C%D1%82%D1%80%D0%B0%D1%84%D0%B8%D0%BE%D0%BB%D0%B5%D1%82%D0%BE%D0%B2%D0%BE%D0%B5_%D0%B8%D0%B7%D0%BB%D1%83%D1%87%D0%B5%D0%BD%D0%B8%D0%B5" title="Ультрафиолетовое излучение">ультрафиолетового излучения</a> и <a href="/wiki/%D0%9C%D0%B5%D1%82%D0%B5%D0%BE%D1%80%D0%B8%D1%82" title="Метеорит">метеоритов</a>.
</p><p>Атмосфера есть у всех массивных тел&#160;— <a href="/wiki/%D0%93%D0%B0%D0%B7%D0%BE%D0%B2%D1%8B%D0%B5_%D0%B3%D0%B8%D0%B3%D0%B0%D0%BD%D1%82%D1%8B" title="Газовые гиганты">газовых гигантов</a>, в <a href="/wiki/%D0%A1%D0%BE%D0%BB%D0%BD%D0%B5%D1%87%D0%BD%D0%B0%D1%8F_%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0" title="Солнечная система">Солнечной системе</a>&#160;— у <a href="/wiki/%D0%9F%D0%BB%D0%B0%D0%BD%D0%B5%D1%82%D1%8B_%D0%B7%D0%B5%D0%BC%D0%BD%D0%BE%D0%B9_%D0%B3%D1%80%D1%83%D0%BF%D0%BF%D1%8B" title="Планеты земной группы">планет земной группы</a>, кроме <a href="/wiki/%D0%9C%D0%B5%D1%80%D0%BA%D1%83%D1%80%D0%B8%D0%B9" title="Меркурий">Меркурия</a>. 
</p>

<p><a href="/wiki/%D0%90%D1%82%D0%BC%D0%BE%D1%81%D1%84%D0%B5%D1%80%D0%BD%D0%BE%D0%B5_%D0%B4%D0%B0%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5" title="Атмосферное давление">Атмосферное давление</a>&#160;— это <a href="/wiki/%D0%A4%D0%B8%D0%B7%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%B2%D0%B5%D0%BB%D0%B8%D1%87%D0%B8%D0%BD%D0%B0" title="Физическая величина">физическая величина</a>, численно равная силе, действующей в атмосфере на единицу <a href="/wiki/%D0%9F%D0%BB%D0%BE%D1%89%D0%B0%D0%B4%D1%8C" title="Площадь">площади</a> поверхности перпендикулярно этой поверхности. Атмосферное давление создаётся <a href="/wiki/%D0%93%D1%80%D0%B0%D0%B2%D0%B8%D1%82%D0%B0%D1%86%D0%B8%D1%8F" title="Гравитация">гравитационным</a> притяжением воздуха к планете. В <a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%B0%D1%8F_%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0_%D0%B5%D0%B4%D0%B8%D0%BD%D0%B8%D1%86" title="Международная система единиц">Международной системе единиц (СИ)</a> давление измеряется в <a href="/wiki/%D0%9F%D0%B0%D1%81%D0%BA%D0%B0%D0%BB%D1%8C_(%D0%B5%D0%B4%D0%B8%D0%BD%D0%B8%D1%86%D0%B0_%D0%B8%D0%B7%D0%BC%D0%B5%D1%80%D0%B5%D0%BD%D0%B8%D1%8F)" title="Паскаль (единица измерения)">паскалях</a>, в средствах массовой информации давление атмосферы характеризуется также в <a href="/wiki/%D0%9C%D0%B8%D0%BB%D0%BB%D0%B8%D0%BC%D0%B5%D1%82%D1%80_%D1%80%D1%82%D1%83%D1%82%D0%BD%D0%BE%D0%B3%D0%BE_%D1%81%D1%82%D0%BE%D0%BB%D0%B1%D0%B0" title="Миллиметр ртутного столба">миллиметрах ртутного столба</a>. При «нормальных условиях», согласно <a href="/wiki/%D0%A1%D1%82%D0%B0%D0%BD%D0%B4%D0%B0%D1%80%D1%82%D0%BD%D0%B0%D1%8F_%D0%B0%D1%82%D0%BC%D0%BE%D1%81%D1%84%D0%B5%D1%80%D0%B0" title="Стандартная атмосфера">Международной стандартной атмосфере</a>, давление <a href="/wiki/%D0%92%D0%BE%D0%B7%D0%B4%D1%83%D1%85" title="Воздух">воздуха</a> на среднем <a href="/wiki/%D0%A3%D1%80%D0%BE%D0%B2%D0%B5%D0%BD%D1%8C_%D0%BC%D0%BE%D1%80%D1%8F" title="Уровень моря">уровне моря</a> при <a href="/wiki/%D0%A2%D0%B5%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D1%83%D1%80%D0%B0_%D0%B2%D0%BE%D0%B7%D0%B4%D1%83%D1%85%D0%B0" title="Температура воздуха">температуре</a> 15&#160;°C принято равным 101325 <a href="/wiki/%D0%9F%D0%B0%D1%81%D0%BA%D0%B0%D0%BB%D1%8C_(%D0%B5%D0%B4%D0%B8%D0%BD%D0%B8%D1%86%D0%B0_%D0%B8%D0%B7%D0%BC%D0%B5%D1%80%D0%B5%D0%BD%D0%B8%D1%8F)" title="Паскаль (единица измерения)">Па</a> или 760 <a href="/wiki/%D0%9C%D0%B8%D0%BB%D0%BB%D0%B8%D0%BC%D0%B5%D1%82%D1%80_%D1%80%D1%82%D1%83%D1%82%D0%BD%D0%BE%D0%B3%D0%BE_%D1%81%D1%82%D0%BE%D0%BB%D0%B1%D0%B0" title="Миллиметр ртутного столба">мм&#160;рт.&#160;ст.</a> <a href="/wiki/%D0%90%D1%82%D0%BC%D0%BE%D1%81%D1%84%D0%B5%D1%80%D0%BD%D0%BE%D0%B5_%D0%B4%D0%B0%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5" title="Атмосферное давление">Давление атмосферы</a> уменьшается с высотой в соответствии с <a href="/wiki/%D0%91%D0%B0%D1%80%D0%BE%D0%BC%D0%B5%D1%82%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%84%D0%BE%D1%80%D0%BC%D1%83%D0%BB%D0%B0" title="Барометрическая формула">барометрической формулой</a>.
</p>
<div class="mw-heading mw-heading2"><h2 id="Состав_атмосферы"><span id=".D0.A1.D0.BE.D1.81.D1.82.D0.B0.D0.B2_.D0.B0.D1.82.D0.BC.D0.BE.D1.81.D1.84.D0.B5.D1.80.D1.8B"></span>Состав атмосферы</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%90%D1%82%D0%BC%D0%BE%D1%81%D1%84%D0%B5%D1%80%D0%B0&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Состав атмосферы»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D1%82%D0%BC%D0%BE%D1%81%D1%84%D0%B5%D1%80%D0%B0&amp;action=edit&amp;section=2" title="Редактировать код раздела «Состав атмосферы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Начальный состав атмосферы планеты обычно зависит от химических и температурных свойств <a href="/wiki/%D0%A1%D0%BE%D0%BB%D0%BD%D1%86%D0%B5" title="Солнце">Солнца</a> в период формирования планет и последующего выхода внешних газов. Затем состав газовой оболочки эволюционирует под действием различных факторов.
</p><p>Атмосферы <a href="/wiki/%D0%92%D0%B5%D0%BD%D0%B5%D1%80%D0%B0_(%D0%BF%D0%BB%D0%B0%D0%BD%D0%B5%D1%82%D0%B0)" class="mw-redirect" title="Венера (планета)">Венеры</a> и <a href="/wiki/%D0%9C%D0%B0%D1%80%D1%81_(%D0%BF%D0%BB%D0%B0%D0%BD%D0%B5%D1%82%D0%B0)" class="mw-redirect" title="Марс (планета)">Марса</a> в основном состоят из <a href="/wiki/%D0%94%D0%B8%D0%BE%D0%BA%D1%81%D0%B8%D0%B4_%D1%83%D0%B3%D0%BB%D0%B5%D1%80%D0%BE%D0%B4%D0%B0" title="Диоксид углерода">диоксида углерода</a> с небольшими добавлениями <a href="/wiki/%D0%90%D0%B7%D0%BE%D1%82" title="Азот">азота</a>, <a href="/wiki/%D0%90%D1%80%D0%B3%D0%BE%D0%BD" title="Аргон">аргона</a>, <a href="/wiki/%D0%9A%D0%B8%D1%81%D0%BB%D0%BE%D1%80%D0%BE%D0%B4" title="Кислород">кислорода</a> и других газов. Земная атмосфера в большой степени является продуктом живущих в ней организмов, так почти весь кислород появился в ней в результате <a href="/wiki/%D0%9A%D0%B8%D1%81%D0%BB%D0%BE%D1%80%D0%BE%D0%B4%D0%BD%D0%B0%D1%8F_%D1%80%D0%B5%D0%B2%D0%BE%D0%BB%D1%8E%D1%86%D0%B8%D1%8F" class="mw-redirect" title="Кислородная революция">кислородной революции</a>. Состав атмосферы <a href="/wiki/%D0%97%D0%B5%D0%BC%D0%BB%D1%8F" title="Земля">Земли</a> (по объёму сухого воздуха): 78,08&#160;% <a href="/wiki/%D0%90%D0%B7%D0%BE%D1%82" title="Азот">азота</a>, 20,95&#160;% <a href="/wiki/%D0%9A%D0%B8%D1%81%D0%BB%D0%BE%D1%80%D0%BE%D0%B4" title="Кислород">кислорода</a>, 0,934&#160;% <a href="/wiki/%D0%90%D1%80%D0%B3%D0%BE%D0%BD" title="Аргон">аргона</a>, 0,034&#160;— 0,042&#160;% <a href="/wiki/%D0%94%D0%B8%D0%BE%D0%BA%D1%81%D0%B8%D0%B4_%D1%83%D0%B3%D0%BB%D0%B5%D1%80%D0%BE%D0%B4%D0%B0" title="Диоксид углерода">диоксида углерода</a> и небольшое количество <a href="/wiki/%D0%92%D0%BE%D0%B4%D0%BE%D1%80%D0%BE%D0%B4" title="Водород">водорода</a>, <a href="/wiki/%D0%93%D0%B5%D0%BB%D0%B8%D0%B9" title="Гелий">гелия</a>, других <a href="/wiki/%D0%91%D0%BB%D0%B0%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B5_%D0%B3%D0%B0%D0%B7%D1%8B" title="Благородные газы">благородных газов</a> и загрязнителей; количество <a href="/wiki/%D0%92%D0%BE%D0%B4%D1%8F%D0%BD%D0%BE%D0%B9_%D0%BF%D0%B0%D1%80" title="Водяной пар">водяного пара</a> сильно зависит от широты и колеблется от 0,2 до 2,5&#160;%,
</p><p>Низкотемпературные <a href="/wiki/%D0%93%D0%B0%D0%B7%D0%BE%D0%B2%D1%8B%D0%B5_%D0%B3%D0%B8%D0%B3%D0%B0%D0%BD%D1%82%D1%8B" title="Газовые гиганты">газовые гиганты</a>&#160;— <a href="/wiki/%D0%AE%D0%BF%D0%B8%D1%82%D0%B5%D1%80_(%D0%BF%D0%BB%D0%B0%D0%BD%D0%B5%D1%82%D0%B0)" class="mw-redirect" title="Юпитер (планета)">Юпитер</a>, <a href="/wiki/%D0%A1%D0%B0%D1%82%D1%83%D1%80%D0%BD_(%D0%BF%D0%BB%D0%B0%D0%BD%D0%B5%D1%82%D0%B0)" class="mw-redirect" title="Сатурн (планета)">Сатурн</a>, <a href="/wiki/%D0%A3%D1%80%D0%B0%D0%BD_(%D0%BF%D0%BB%D0%B0%D0%BD%D0%B5%D1%82%D0%B0)" title="Уран (планета)">Уран</a> и <a href="/wiki/%D0%9D%D0%B5%D0%BF%D1%82%D1%83%D0%BD_(%D0%BF%D0%BB%D0%B0%D0%BD%D0%B5%D1%82%D0%B0)" class="mw-redirect" title="Нептун (планета)">Нептун</a>&#160;— могут удерживать газы с низкой <a href="/wiki/%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D1%8F%D1%80%D0%BD%D0%B0%D1%8F_%D0%BC%D0%B0%D1%81%D1%81%D0%B0" title="Молекулярная масса">молекулярной массой</a>&#160;— <a href="/wiki/%D0%92%D0%BE%D0%B4%D0%BE%D1%80%D0%BE%D0%B4" title="Водород">водород</a> и <a href="/wiki/%D0%93%D0%B5%D0%BB%D0%B8%D0%B9" title="Гелий">гелий</a>.
Высокотемпературные газовые гиганты, такие как <a href="/wiki/%D0%9E%D1%81%D0%B8%D1%80%D0%B8%D1%81_(%D0%BF%D0%BB%D0%B0%D0%BD%D0%B5%D1%82%D0%B0)" class="mw-redirect" title="Осирис (планета)">Осирис</a> или <a href="/wiki/51_%D0%9F%D0%B5%D0%B3%D0%B0%D1%81%D0%B0_b" title="51 Пегаса b">51 Пегаса b</a>, наоборот, не могут её удержать и <a href="/wiki/%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D0%B0" title="Молекула">молекулы</a> их атмосферы <a href="/wiki/%D0%94%D0%B8%D1%81%D1%81%D0%B8%D0%BF%D0%B0%D1%86%D0%B8%D1%8F_%D0%B0%D1%82%D0%BC%D0%BE%D1%81%D1%84%D0%B5%D1%80_%D0%BF%D0%BB%D0%B0%D0%BD%D0%B5%D1%82" title="Диссипация атмосфер планет">рассеиваются</a> в пространстве. Этот процесс протекает медленно, постоянно.
</p>
